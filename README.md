# BlenderGuadua
## v0.1 | 24.07.11

Setup de GeometryNodes en Blender para crear guaduas y bambúes paramétricos. Incluye opciones para:
- Largo y ancho de la guadua.
- Número de nodos.
- Cortes y control de profundidad y ángulo.
- Extrusión (espesor de la guadua)
